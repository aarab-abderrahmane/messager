Getting Started — Developer Setup
Developer Setup

Get MSN Chat running locally in under 2 minutes.

1. Clone the repository

bash 
# Clone the front-end
git clone https://github.com/aarab-abderrahmane/messager.git
cd messager/frontEnd

2. Install dependencies

bash 
npm install   # installs React, Tailwind, Framer Motion, etc.

3. Configure environment

.env 
VITE_API_URL=http://localhost:3000      # back-end base URL
VITE_GIPHY_API_KEY=your_giphy_key      # Giphy API key for GIF picker
VITE_SOCKET_URL=http://localhost:3000  # Socket.io server URL

4. Run the dev server

bash 
npm run dev   # starts Vite on http://localhost:5173

UI Component Reference — Help
Component Reference

Key props for the main reusable UI components.

TitleBar

PropTypeDefaultDescription
titlestring—Window title text displayed in the blue bar
iconReactNode—Optional icon before the title
onClose() => void—Callback for the ✕ close button
showControlsbooleantrueShow min/max/close window controls

MessageBubble

PropTypeDefaultDescription
messageMessage—Full message object (text, sender, timestamp, type)
isOwnbooleanfalseRight-align and use accent colour for own messages
onReact(emoji) => void—Triggered when user picks an emoji reaction
onReply(msg) => void—Triggered when user clicks "Reply"

InputArea

PropTypeDefaultDescription
onSend(content) => void—Fires with message payload on submit
onTyping() => void—Debounced callback to broadcast typing event
disabledbooleanfalseDisables input & send button when not connected

StickerPicker

PropTypeDefaultDescription
onSelect(gif: GiphyResult) => void—Returns selected GIF/sticker object
apiKeystringenv varGiphy API Key (falls back to VITE_GIPHY_API_KEY)
defaultTab"gifs" | "stickers""gifs"Which tab is active on open

VoiceMessage

PropTypeDefaultDescription
srcstring—Blob URL or remote URL of the audio clip
durationnumber—Duration in seconds for the progress bar
waveformnumber[]—Array of amplitude values for visualiser bars
